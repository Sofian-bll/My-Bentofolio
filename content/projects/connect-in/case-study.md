## Contexte

Réseau social complet : authentification par tokens Sanctum, fil d’actualité, relations many-to-many (follows, likes), et une suite de tests unitaires couvrant les endpoints critiques. Frontend Vue 3 avec composants shadcn-vue.

## Points clés

- API REST sécurisée par tokens Sanctum (auth, fil d’actualité).
- Relations many-to-many (follows, likes) sous MySQL avec tests.
- Architecture full-stack Laravel + Vue 3 pensée pour scaler.

## Stack

- Laravel
- Sanctum
- MySQL
- Vue 3
