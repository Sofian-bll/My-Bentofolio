## Contexte

Expérimentation d’une bibliothèque de composants UI générés par IA ne nécessitant aucun JavaScript côté client. Une approche radicale de la performance web.

## Points clés

- Composants UI fonctionnels sans JavaScript côté client.
- Génération de code par Gemini AI.
- Approche radicale de performance web.

## Stack

- TypeScript
- Gemini AI
