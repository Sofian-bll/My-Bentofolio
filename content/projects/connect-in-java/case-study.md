## Contexte

Deuxième itération du réseau social Connect’IN, porté de Laravel (PHP) vers Java. Fonctionnalités identiques (authentification, fil d’actualité, follows, likes) réimplémentées dans une stack Java + Vue 3.

## Points clés

- Migration complète du backend Laravel vers Java.
- Mêmes fonctionnalités : auth, follows, likes, fil d’actualité.
- Architecture fullstack Java + Vue 3.

## Stack

- Java
- Vue 3
- MySQL
