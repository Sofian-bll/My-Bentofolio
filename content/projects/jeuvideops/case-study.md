## Contexte

Projet DevOps/DevSecOps complet : pipeline CI/CD automatisé avec gates de PR, tests E2E Playwright, rapports Allure, et déploiement continu sur GitHub Pages. Automatiser plus pour travailler moins.

- Badges CI visibles, workflow DevSecOps complet.
